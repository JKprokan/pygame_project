<h1>블록깨기 게임</h1>

- <h3>기능</h3>   
->  1. 패들에서 공을 날려서 벽 또는 블럭과 부딪히면 튕겨나간다. <br/>
    2. 블럭과 부딪히면 블럭이 사라진다. <br/>
    3. 목숨은 3개이고 블럭을 다 없애면 Clear, 목숨이 다 사라지면 Fail <br/>
     <br/>
